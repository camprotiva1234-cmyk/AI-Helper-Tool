Claude is an AI assistant you talk to by typing. A model is the "brain" you're talking to — Haiku is the fast, cheap one; Sonnet is the balanced all-rounder; Opus is the slow, powerful one for hard problems. Claude comes in three apps: Chat is the website (claude.ai — nothing to install, start here), Cowork is a desktop app that can create and edit real files on your computer, and Code is a tool for building software. This planner tells you which combination to use for your project and writes the exact message to paste in — and each step includes plain-language directions for getting there.
